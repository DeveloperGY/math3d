# m3::vec4::normalized()

## Signature

```c++
namespace m3
{
    static m3::vec4 struct vec4::normalized(const m3::vec4 &v);

```

## Description

Calculates and returns the normalized version of the specified m3::vec4.

### Parameters

`void`: The specified m3::vec4

### Return Value

`void`: Nothing
